# simpleETLpipeline
Simple ETL pipeline in Python that extracts CSV/JSON/XML, transforms units, and loads consolidated data

Goal:
Build a simple ETL (Extract–Transform–Load) process in Python that collects height and weight data from multiple formats (CSV, JSON, XML), converts units from imperial to metric, and consolidates them into a single CSV file for AI model training.

Technologies

Python 3.11

pandas

xml.etree.ElementTree

datetime, glob
